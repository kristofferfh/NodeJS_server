# Assignment: NodeJS server
Requirments:
* [NodeJS](https://nodejs.org/en)
* [MongoDB (local or cloud)](https://www.mongodb.com/)
* API caller

To test this server clone repo to folder, then use `npm install`

remember to create an .env file containing
```
ACCES_TOKEN_SECRET = *INT*
REFRESH_TOKEN_SECRET = *INT*
DATABASE_URI = *URI*
```
